# 微风企机构数统计

用于处理微风企每周/每月机构数和企业数统计数据，生成标准化的 Excel 报表。

## 输入文件

需要以下 Excel 文件，放技能的同目录下：

### 世荣提供的表
- 3月企业用户数.xlsx
- 3月预充值.xlsx
- 3月历史客户.xlsx
- 3月月结用户.xlsx
- 3月机构用户.xlsx
- 3月全部历史客户.xlsx

### 财务提供的表
- 客户数据汇总表-市场部26.4.xls
- 客户数据汇总表-战略增长中心26.4.xls
- 回款数据-需去重26.4.1.xls

### 模板文件
- **模板.xlsx**：需要用户提供的模板文件

## 用户输入

用户会提供：
1. **调用时间**（如"3月"、"3月第1周"等）- 填充到E列和L列
2. **数据文件和模板** - 放在同目录下

## 输出文件

生成以下文件：
1. 2026年X月未与财务核对版本.xlsx - 主输出文件
2. 机构用户数-整理.xlsx
3. 补贴比例统计.xlsx
4. 结果a.xlsx
5. 结果b.xlsx
6. 结果c.xlsx

## 处理逻辑

### 步骤1-2: 读取财务表并填充模板
- 从财务的2个表格读取：部门、组别、姓名
- 写入模板对应列（B、C、D列）
- A列填充序号（从1开始）
- **E列和L列填充用户提供的调用时间**（如"3月"）
- 行高统一为15
- 所有单元格左对齐
- **注意：模板中72-84行的原有格式和说明文字必须保留，不能删除**

### 步骤5: vlookup企业用户数
- 从"企业用户数"表获取 姓名→企业客户数 的映射
- 用vlookup填充到模板F列
- **F列必须转换为数字类型（int），不能是字符串**
- 如果找不到，对应F列显示0，不删除行

### 步骤6-9: 处理机构用户数（结果a）
- 复制原始"机构用户数"表
- 添加3个新列：认证企业、认证企业2、企业名称
- 列17: 复制列9
- 列18: 从"预充值"表按行号复制认证企业（第8列）
- 列19: 从"历史客户"表按行号复制企业名称（第5列）

### 步骤10-11: 去重（结果b）
- 如果第17列的值在17/18/19三列中有重复，删除该行

### 步骤12-13: 回款数据去重（结果c）
- 根据用户提供的回款数据筛选对应周期的手机号
- 删除机构用户中手机号匹配的行

### 步骤15: 补贴比例统计
- 基于结果c统计每个用户的补贴比例计数

### 步骤16: vlookup填充补贴比例
- 列18: 50%研发数据
- 列19: 10%比例发放机构用户数
- 找不到设为0

### 步骤17: 历史客户调用数
- 统计"历史客户"表中每个用户的数据条数
- 填充到模板列22，找不到设为0

## 步骤18-24: 生成与财务核对版

从"与财务核对版"模板执行以下操作：

### 步骤18: 复制基础数据
- 将"未与财务核对版本"中的第1-6列和第12列复制到"模版2"的对应列

### 步骤19-21: 复制补贴数据
- R列 → T列
- S列 → U列  
- V列 → X列

### 步骤22: 财务表P列匹配
- 从"客户数据汇总表-市场部"和"客户数据汇总表-战略增长中心"的P列获取数据
- 根据D列的姓名匹配，填充到模版2的P列

### 步骤23: 财务表R列匹配
- 从"客户数据汇总表-市场部"和"客户数据汇总表-战略增长中心"的R列获取数据
- 根据D列的姓名匹配，填充到模版2的R列

### 步骤24: 计算列O和列M
- **列O = 列P + 列R + 列X**（列O是第15列）
- **列M = 列O**（列M是第13列）

### 步骤25: 计算列R
- **列R = 列S + 列T**（列R是第18列）

### 步骤23: 财务表R列匹配
- 从两个财务表的R列获取数据
- 根据D列的姓名匹配，填充到模版2的S列

### 步骤24: 保存
- 将模版2另存为"与财务核对版.xlsx"
